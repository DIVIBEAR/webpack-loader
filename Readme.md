## 目录结构

```
├── dist         输出文件
├── loaders         自定义loader
├── plugins        自定义插件
├── src          模块等资源文件
├── node_modules  依赖包
├── package.json
├── README.md       说明文档
├── webpack.config.js       webpack配置文件
└── .gitignore
```
